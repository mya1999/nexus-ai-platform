# nexus-ai-platform

ملخص
مشروع واجهة Next.js لعرض معاينات التصميم مع شريط تقدم ونماذج API بسيطة.

التشغيل محلياً
1. انسخ المستودع: git clone https://github.com/mya1999/nexus-ai-platform.git
2. ادخل المجلد: cd nexus-ai-platform
3. ثبت الحزم: npm install
4. شغّل في وضع التطوير: npm run dev
5. افتح: http://localhost:3000/designs/1
